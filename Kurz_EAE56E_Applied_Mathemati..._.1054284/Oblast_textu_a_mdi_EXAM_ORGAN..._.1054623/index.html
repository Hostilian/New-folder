Exam organization

The written test

It takes 100 minutes to complete the test, and you must achieve at least 60% of the points.

You will write the test in the exam mode in Moodle, which has many limitations resulting from the need to ensure the correctness of the exam tests.

What are the limitations of this exam mode?

•         In terms of programs suitable for us, only Moodle and Excel (and calculator) are available (and several other programs according to the wishes of other teachers as statistical programs, etc.).

•         Word is unavailable.

•         You cannot save any file so it cannot be uploaded as a reply.

•         Only Excel files can be opened in Moodle.

What does this mean for you?

As you already know, the tests are designed for the Open book mode, so this means you can use mainly paper materials, books, your notes, etc. You definitely cannot use mobile phones, tablets, your own computers.

Within the test, there are several questions where you do not bother with the format of the answer; these are questions where you choose the correct answer, write one value, choose the appropriate word, etc.

However, there is one type of question - a question with a written answer, where you have to write your answer, most often some text, but also a graph or picture in the answer window. Follow these steps:

•         If you need to write a verbal answer, use the editor in the answer window.

•         If you need to paste a table or chart, create it in Excel, copy and paste.

•         If you need to insert a picture, add a Draw tab in Excel in the Options - Customize Ribbon menu. Then you can create simple diagrams and pictures. Use Lasso or Select Objects in Find and Select to select the image and copy and paste it into the answer window. You can try these procedures in the Small test for practice in writing answers, which you will find in the section describing the course and conditions of the exam on the subject page.
