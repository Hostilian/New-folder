Popis

This test contains simple tasks for practicing possible ways of answering. It consists of a solution of a system of linear equations with generated coefficients, so the result may look pretty ugly.

If you have to write your answer, most often some text, but also a graph or picture in the answer window, follow these steps:

•         If you need to write a verbal answer, use the editor in the answer window.

•         If you need to paste a table or chart, create it in Excel, copy and paste.

•         If you need to insert a picture, add a Draw tab in Excel in the Options - Customize Ribbon menu. Then you can create simple diagrams and pictures. Use Lasso or Select Objects in Find and Select to select the image and copy and paste it into the answer window. You can try these procedures in the Small test for practice in writing answers, which you will find in the section describing the course and conditions of the exam on the subject page.


You can solve this test as many times as you like, or you can also send me a request to approve a way of writing the answer or any other questions, also technical.
