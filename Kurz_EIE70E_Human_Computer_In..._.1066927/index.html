Example of UseCase:

Use Case is description of user goal. For this subject purpose we will use the words Expect and Wants, which will be describing the user needs for UI. UseCases we are deriving from Goals. UseCases are written from the user point of view.

Example: bicycle navigation
Goal: find the shortest way for the bike

Use Case: Searching the shortest distance between towns
User Expect: the system give him possibility to put start town and put final town, he expect the system shows the:

  • shortest way
  • way which goes thru forest
  • way which goes via asfalt route
  • way which follows the river


The user expect the syste show the list of possibilities and give him chane to choose one.

UseCase: choose  favourite route

The user choose the route from the list of rotes and expect, the system show him:

  • map and route placed on the map
  • information about distance between start and finial point
  • avg. speed

UseCase: Put pin on the route

The user wants to put pin on the route. He wants to use pin symbol and put it into the map by touch at the navigation screen
